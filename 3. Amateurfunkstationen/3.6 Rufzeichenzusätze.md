## 3.6 Rufzeichenzusätze
> 28.04.26 / [Source](https://50ohm.de/N_rufzeichenzusaetze.html) / [Video](https://www.youtube.com/watch?v=E5RuFuUv7ag&t=890s)

### Wie wird Rufzeichenzusätze eingebaut?
- Ein Rufzeichenzusatz wird mit einem **Schrägstrich /** an das Rufzeichen gehängt.
- Man spricht den **/** als **Stroke** aus.

### Zusätze:
| Zusatz  | Gesprochen         | Bedeutung                                                   |
| ------- | ------------------ | ----------------------------------------------------------- |
| **/am** | aeronautisch mobil | an Bord eines Luftfahrzeugs im Flug.                        |
| **/mm** | maritim mobil      | an Bord eines Schiffs auf See, außerhalb der 12-Meilen-Zone |
| **/m**  | mobil              | aus Landfahrzeug oder Schiff auf Binnengewässern            |
| **/p**  | portabel           | zu Fuß unterwegs oder vorübergehend ortsfest                |
| **/R**  | Remote             | Remote-Betrieb                                              |
| **/T**  | Trainee            | Ausbildungsfunk                                             |
- Bei **/am und /mm** muss der **Schiff / Luftfahrzeug-führer** den Funkbetrieb erlauben.
	- Es ist **keine** Sondergenehmigung erforderlich.
- In Deutschland muss **/p** nicht zwingend benutzt werden. Es ist freiwillig.
