## 3.8 Fernbediente und automatische Stationen
> 28.04.26 / [Source](https://50ohm.de/N_fernbediente_automatische_stationen.html) / [Video](https://www.youtube.com/watch?v=E5RuFuUv7ag&t=1405s)

### Was bedeutet „besetzt“ und „unbesetzt“?
- Eine normale Funkstation wird **besetzt** betrieben.
	- **Unmittelbar**: Funkamateur ist **direkt** an der Station
	- **Mittelbar**: Funkamateur kontrolliert die Station aus der **Ferne**, z. B. bei Remote-Betrieb

### Was ist eine Relaisfunkstelle (Repeater)?
- Sie **empfängt** auf einer Frequenz und sendet das Empfangene auf einer **anderen Frequenz wieder aus**.
- Dadurch kann man auf **größere Entfernungen** funken.

### Was ist eine Bake (Beacon)?
- Es ist ein **automatischer** Sender.
- Sie sendet regelmäßig ein Signal, oft nur:
	- ihr **Rufzeichen**,
	- ein **Testsignal**,
	- manchmal **technische Informationen**.

### Was ist wichtig für Relais oder Bake?
- Man braucht eine **besondere Rufzeichenzuteilung** für eine automatisch oder fernbedient arbeitende Station.
- Der **Standort** wird in der Zuteilungsurkunde **festgelegt**.
