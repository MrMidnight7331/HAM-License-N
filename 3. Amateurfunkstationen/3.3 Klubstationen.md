## 3.3 Klubstationen
> 27.04.26 / [Source](https://50ohm.de/N_klubstationen.html) / [Video](https://www.youtube.com/watch?v=E5RuFuUv7ag&t=363s)

### Was sind Klubstationen?
- Eine **Klubstation** wird von **mindestens 3 Funkamateure** gemeinsam betrieben. 
- Beispiel:
	- einem **DARC-Ortsverband**
	- einer **Schul-AG**
	- einer **Funkgruppe**
	- einem **Ausbildungsstandort**.
-  Eine Klubstation darf **jeder** Funkamateur nutzen, der eine **gültige Zulassung** hat.
- Eine Klubstation darf **kurzzeitig** an einem anderen Standort betrieben werden.
	- Dies muss **nicht** bei der BNetzA gemeldet werden.
- **Klubstationsrufzeichen** haben bestimmte Bereiche im Rufzeichenplan.
- Beispiel:

| Rufzeichen |     |        | Klasse |
| ---------- | --- | ------ | ------ |
| DAØAA      | -   | DAØZZZ | A      |
| DAØA       | -   | DA3Z   | A      |
| DBØA       | -   | DD9Z   | A      |
| DFØA       | -   | DH9Z   | A      |
| DJØA       | -   | DM9Z   | A      |
| DFØAA      | -   | DFØZZZ | A      |
| DKØAA      | -   | DKØZZZ | A      |
| DLØAA      | -   | DLØZZZ | A      |
| DNØA       | -   | DNØZ   | E      |
| DNØAA      | -   | DNØZZZ | E      |
| DOØA       | -   | DO9Z   | E      |
| DP3A       | -   | DP9Z   | A      |
| DQØA       | -   | DR9Z   | A      |
- Viele Klubstationen haben eine **0** im Rufzeichen, es ist aber **nicht** ausschlaggebend!

### Voraussetzungen für ein Klubstationsrufzeichen:
1. Mindestens **3** Funkamateure
2. Eine **Stationsverantwortlicher**
3. Stationsverantwortlicher **muss selbst** Funkamateur sein
4. **Zuteilung** durch BNetzA
5. Zulassung zur Teilnahme nach **§ 3 Abs. 1 AFuG**

- Die Zuteilung ist**5 Jahre** befristet. Danach muss es verlängert werden.
- Der Stationsverantwortlicher wird **Inhaber** des Klubrufzeichens.

### Welche Klasse gilt beim Senden?
- Es gilt immer die **niedrigere** Klasse.

| Funkamateur | Klubstation      | Erlaubt ist maximal |
| ----------- | ---------------- | ------------------- |
| Klasse N    | Station Klasse A | nur Klasse N        |
| Klasse A    | Station Klasse N | nur Klasse N        |
