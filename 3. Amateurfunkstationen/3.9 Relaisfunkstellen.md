## 3.9 Relaisfunkstellen
> 29.04.26 / [Source](https://50ohm.de/N_relaisfunkstellen.html / [Video](https://www.youtube.com/watch?v=E5RuFuUv7ag&t=1419s) [Video](https://www.youtube.com/watch?v=E5RuFuUv7ag&t=1419s)

### Was sind Relais?
- Es **empfängt** ein Signal und **sendet es wieder aus**.
- Ermöglicht **größere Reichweite** als direkter Verbindungen
- Ein Relais steht **oft** an einem **hohen** Standort
- Relaisfunkstellen werden auch kurz als Relais oder als Repeater bezeichnet.

### Woran erkennt man Relaisfunkstellen?
- Sie senden regelmäßig ihr **Rufzeichen** aus.
	- Typische Rufzeichen beginnen oft mit:
		- **DB0**
		- **DM0**
		- **DO0**
	- Relais werden **nicht** mit normalen persönlichen Rufzeichen betrieben.
	- Sie sind üblicherweise **nicht ständig besetzt**.
- Relais haben beistens 2 Frequenzen:
	- **Eingabefrequenz**
	- **Ausgabefrequenz**
- Der **Abstand** zwischen beiden Frequenzen heißt **Ablage**.

### Ablagen in Deutschland:

| Band  | Übliche Ablage |
| ----- | -------------: |
| 10 m  |        100 kHz |
| 2 m   |        600 kHz |
| 70 cm |        7,6 MHz |
| 23 cm |         28 MHz |
Beispiel am 70cm Relai:
- Ausgabefrequenz: **438,875 MHz**
- Ablage: **−7,600 MHz**
- Eingabefrequenz: **431,275 MHz**

Merke: Ablage **immer niedriger** als **Ausgabefrequenz**

### Crossband-Betrieb
- Ein Relais kann zwei **verschiedene Bänder verbinden**.
- Beispielsweise:
	- eine Station arbeitet auf **70 cm**
	- eine andere auf **2 m**
- Manchmal kann dabei auch die Sendeart **umgesetzt** werden, z. B. von **SSB** auf **FM**.

### Digipeater
- Ein **Digipeater** (Daten-Repeater) ist ein Relai für **Daten**.
- Es kann:
	- Datenpakete **empfangen**,
	- Datenpakete wieder **aussenden**,
	- Teile **wiederholen oder verändern**.
- Es kann aber **nur**:
	- Datenpakete **zeitversetzt**,
	- oder **in Teilen** aussenden.

### Verfahren bei Relais:
- Bei **nalogen VHF-/UHF-Relais** für **Sprache** nutzt man meistens **FM**(Frequenzmodulation).
- **Digitale** Sprachverfahren benutzt man meistens **DMR, D-Star (FM-Sprechfunk)**.

### Narrow-FM vs Wide-FM:
- Bei FM gibt es verschiedene Bandbreiten:
	- FM-W (Wide-FM): **25kHz**
	- FM-N (Narrow-FM): **12,5kHz**
- Viele Relais **erwarten** Narrow-FM.
	- Denn wenn der Signal **zu breit** ist, kann es:
		- verzerrt klingen,
		- Nachbarfrequenzen stören,
		- andere Rlais stlören.

### Verhalten auf Relais
| Regeln                                    | Warum?                                                                 |
| ----------------------------------------- | ---------------------------------------------------------------------- |
| Durchgänge kurz halten                    | mobile/portable Stationen sollen auch drankommen                       |
| Pausen lassen                             | andere könnte sich melden                                              |
| Nicht doppeln (nicht gleichzeitig senden) | gleichzeitiges Sprechen stört                                          |
| Saubere Übergabe                          | damit klar ist, wer als Nächstes spricht. Verhinder damit auch Doppeln |

### Besonderheit bei Relais:
- Für **automatisch** arbeitende Stationen oberhalb von **30 MHz** gilt maximal **50 W ERP** (Effective Radiated Power).
- Beim **Rapport (Signal-report)** über Relais nur: **Lesbarkeit (R)**
	- Es ist **nicht** die Signalstärke des Funkpartners!
	- Denn Signalstärke (S) bezieht sich **auf das Relai**.
